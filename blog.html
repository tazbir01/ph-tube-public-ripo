  • Discuss the scope of var, let, and const.

    Ans: Scope means where variables are available for use. Lets discuss about the scope of var, ler and const.
    var scope: var scope is global scop. Global scop means var can declared outside a function. Var is also function scoped when it declared within a finction. We can redeclare var variables and update variables. var variables are hoisted to the top of their scop.
    let scope: let variable is block scoped. Block scope is inside of second brackets '{}'. That meana let variables only available inside of block. We can not redeclare let but let can be updated.
    const scope: const variable is also block scoped. But const cannot be updated or re-declared.
